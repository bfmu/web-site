# Backend - web-site

API NestJS para el blog personal. Construido con MongoDB, autenticación JWT, OAuth y Swagger.

## Inicio rápido

```bash
npm install
npm run start:dev
```

## Documentación

Toda la documentación del proyecto está centralizada en la carpeta `/docs`:

- **Inicio**: Ver [docs/README.md](../docs/README.md)
- **Blog API**: Ver [docs/api/blog.md](../docs/api/blog.md)
- **Swagger**: http://localhost:3000/api/docs (cuando el backend está corriendo)
